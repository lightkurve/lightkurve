About Lightkurve#

These pages will help you with how to install and cite Lightkurve, and you can see our changelog. We also show some other software tools that rely on or work well with Lightkurve.

If you encounter a problem with Lightkurve, we encourage you to open an issue on the GitHub repository.

If you would like to propose a change or bug fix to Lightkurve, you can learn more in our development documentation.
