Cutting out Target Pixel Files from Kepler Super Stamps or TESS FFIs¶

You can use lightkurve to cut Target Pixel Files out of a series of standard astronomical images, such as K2 Superstamp Mosaics or TESS Full-Frame-Images (FFIs). This brief tutorial will demonstrate how!

In [2]:

%matplotlib inline
from lightkurve import KeplerTargetPixelFile
from glob import glob
import numpy as np

You can grab your file names from wherever you have saved them. Note: We do not recommend you run from the USB stick provided as the read-write speed is slow.) lightkurve assumes the files are given in time order. So we’ll sort the filenames first.

In [23]:

fnames = np.sort(glob('../data/Kepler-Cluster/Q1/*.fits'))

Now we use the KeplerTargetPixelFile class and the function from_fits_images() to create the new TPF. This will cut out around the position keyword. You can pass a pixel position in units of the original 200x200 super stamp or RA and Dec coordinates.

In [24]:

tpf = KeplerTargetPixelFile.from_fits_images(images=fnames,
                                             position=(95,60),
                                             size=(5,5))

You can write out your new target pixel file easily using to_fits().

In [25]:

tpf.to_fits(output_fn='myTPF.fits', overwrite=True)

Now your KeplerTargetPixelFile can use all of the methods from lightkurve. For example you can plot your TPF

In [26]:

tpf.plot()

Out[26]:

<matplotlib.axes._subplots.AxesSubplot at 0x1c21247d30>

There will also be an aperture in your new TPF you can use for aperture photometry. Let’s take a look

In [27]:

tpf.plot(aperture_mask=tpf.pipeline_mask)

Out[27]:

<matplotlib.axes._subplots.AxesSubplot at 0x1c212d8cc0>

Unfortunately by cutting the superstamp into a target pixel file our aperture is not correct. But you can create your own aperture using boolean arrays:

In [28]:

aper = ~tpf.pipeline_mask
aper[2:4, 2:4] = True
tpf.plot(aperture_mask=aper)

Out[28]:

<matplotlib.axes._subplots.AxesSubplot at 0x1c21e38908>

You can now pass that new aperture to the to_lightcurve() function. This will create a new KeplerLightCurve object which let’s you access even more useful functions.

In [29]:

lc = tpf.to_lightcurve(aper)

In [30]:

lc.plot()

Out[30]:

<matplotlib.axes._subplots.AxesSubplot at 0x1c2286d2b0>

There are a set of useful functions in KeplerLightCurve objects which you can use to work with the data. These include: * flatten(): Remove long term trends using a Savitzky–Golay filter * remove_outliers(): Remove outliers using simple sigma clipping * remove_nans(): Remove infinite or NaN values (these can occur during thruster firings) * fold(): Fold the data at a particular period * bin(): Reduce the time resolution of the array, taking the average value in each bin.

We can use these simply on a light curve object, for example:

In [31]:

lc.flatten().remove_outliers().plot()

Out[31]:
